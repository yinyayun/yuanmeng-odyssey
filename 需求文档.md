# 项目背景与目标
女儿（元宵）今年4年级，该系统目标为守护其成长，包含如下模块：
1）时光存折：通过学习、运动、劳动等方式争取时间积分，时间积分可用于进行游戏、电视等时间消费
2）成长计划：以拓展其知识面为目标，展示不同阶段培养计划以及每期详细的内容
3）数学培养计划：以冲刺重点初中培养其对数学的兴趣为主，展示数学试卷

# 模块设计
## 时光存折
1、账户开户
2、兑换规则配置：
- 配置不同事项（作业、劳动、运动）投入时间与积分兑换比例，比如作业时间与积分兑换比例，10分钟可兑换5积分
- 配置不同事项（游戏、电视）积分兑换时间的比例，如10分钟电视需要20积分
3、积分存取模块：
- 积分存入：存入不同事项的投入时间，自动兑换积分存入账户
- 积分提取：记录不同消耗事项，自动从账户扣取对应的积分
4、账户预览：账户当前剩余积分，最近一周（可选择时间）的积分兑换和消耗统计，折算当前积分可以兑换的不同事项的是时间（提供折算工具）
5、存取明细查询

## 成长计划，可以命名为智慧树洞，这样让小孩更加有兴趣
1、事先会将每个阶段的成长计划对应html保存在某个目录下，以4年级下学期为例，包含：
- 4年级下学期成长计划预览.html
- 20260404期内容.html
2、该系统只需要支持配置指定根目录，查看不同阶段不同期html即可
3、如本地目录结构如下，在该系统中可以支持按文件结构展示，并查看html内容
```
|--growth_plans/
|--|--| 4年级第二学期/
|--|--|--| 4年级第二学期成长计划总览.html
|--|--|--| 20260405期内容.html
```
## 数学培养计划，也可以叫数能充电站，这样让小孩更加有兴趣
与成长计划模块类似，事先将数学试卷以pdf或者word放入某个目录下，按文件目录方式支持预览下载
