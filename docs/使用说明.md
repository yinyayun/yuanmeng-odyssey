# 元梦大陆 - 使用说明

## 系统简介

**元梦大陆**是专为元宵设计的成长守护系统，包含三大趣味模块：

- **时光银行**：通过学习和劳动赚取积分，兑换游戏娱乐时间
- **智慧树洞**：浏览成长计划和学习资料
- **数能充电站**：查看和下载数学试卷

---

## 快速开始

### 1. 本地开发环境启动

```bash
# 安装依赖
npm run install:all

# 启动开发服务器（同时启动前后端）
npm run dev
```

访问 http://localhost:3000

### 2. 生产环境部署

```bash
# 在阿里云服务器上执行
sudo bash deploy.sh
```

## 模块详解

### 时光银行

#### 赚取积分
1. 进入「积分存取」页面
2. 选择事项（如：写作业、阅读、运动）
3. 输入投入时间（分钟）
4. 系统自动计算可获得的积分
5. 点击「确认存入」

#### 消费积分
1. 在「积分存取」页面切换到提取
2. 选择想要的活动（如：看电视、玩游戏）
3. 输入想要的时间
4. 系统显示需要扣除的积分
5. 确认余额充足后点击「确认提取」

#### 兑换规则配置
- 在「兑换规则」页面可以添加/删除规则
- 赚取规则：X分钟 = Y积分
- 消费规则：X积分 = Y分钟

### 智慧树洞

1. 将成长计划 HTML 文件放入 `backend/uploads/growth_plans/` 目录
2. 按学期/日期组织文件夹结构
3. 在系统中按目录树浏览
4. 点击文件即可预览内容

示例结构：
```
growth_plans/
├── 4年级第二学期/
│   ├── 4年级第二学期成长计划总览.html
│   └── 20260405期内容.html
└── 5年级第一学期/
    └── ...
```

### 数能充电站

1. 将数学试卷 PDF/Word 放入 `backend/uploads/math_papers/` 目录
2. 在系统中浏览文件树
3. 支持在线预览 PDF
4. 点击下载按钮保存到本地

## 数据备份

### 自动备份
系统支持两种备份方式：

1. **GitHub 同步**：数据库自动同步到 GitHub
2. **本地备份**：保留最近 10 个备份文件

### 手动备份
```bash
# 本地备份
curl -X POST http://localhost:3001/api/backup/local

# GitHub 同步
curl -X POST http://localhost:3001/api/backup/git-sync
```

## 技术栈

- **前端**：Vue 3 + Element Plus + Vite
- **后端**：Node.js + Express
- **数据库**：SQLite
- **部署**：Nginx + PM2

## 常见问题

### 1. 如何修改默认账户名称？
在数据库中修改 `accounts` 表的 `name` 字段，或重新创建账户。

### 2. 积分计算错误？
检查「兑换规则」中的时间和积分比例设置是否正确。

### 3. 文件上传后看不到？
确保文件放在正确的目录下，并刷新页面。

### 4. 如何重置数据？
删除 `backend/database/yuanmeng.db` 文件，重启服务会自动重建。
