# 元梦大陆 - 使用说明

## 系统简介

**元梦大陆**是专为元宵设计的成长守护系统，包含三大趣味模块：

- **时光银行**：通过学习和劳动赚取积分，兑换游戏娱乐时间
- **智慧树洞**：浏览成长计划和学习资料
- **数能充电站**：查看和下载数学试卷

### 多角色权限系统

系统支持三种角色，不同角色拥有不同权限：

| 角色 | 用户名 | 默认密码 | 权限说明 |
|------|--------|----------|----------|
| 👨 爸爸 | dad | dad123 | 全部权限（管理后台 + 积分操作） |
| 👩 妈妈 | mom | mom123 | 全部权限（管理后台 + 积分操作） |
| 👧 元宵 | yuanxiao | yuanxiao123 | 查看权限（积分查询 + 存取操作） |

**首次使用请访问登录页面：** `http://你的服务器IP:9999/login`

## 快速开始

### 1. 本地开发环境启动

```bash
# 安装依赖
npm run install:all

# 启动开发服务器（同时启动前后端）
npm run dev
```

访问 http://localhost:3000

### 2. 生产环境部署

#### 方式一：使用部署脚本（推荐）

**首次安装部署：**

```bash
# 1. 执行安装（包含系统更新、依赖安装、代码拉取、构建）
sudo bash deploy.sh install

# 2. 查看 Nginx 配置参考
sudo bash deploy.sh nginx

# 3. 手动配置 Nginx 后，启动后端服务
sudo bash deploy.sh start
```

**部署脚本命令说明：**

| 命令 | 说明 |
|------|------|
| `install` | 首次安装（系统更新 + 依赖安装 + 代码拉取 + 构建） |
| `update` | 更新代码并重新构建 |
| `start` | 启动后端服务 |
| `restart` | 重启后端服务 |
| `stop` | 停止后端服务 |
| `status` | 查看服务状态 |
| `logs` | 查看后端日志 |
| `nginx` | 显示 Nginx 配置参考 |

**日常更新：**
```bash
sudo bash deploy.sh update
```

**服务管理：**
```bash
sudo bash deploy.sh restart   # 重启服务
sudo bash deploy.sh logs      # 查看日志
sudo bash deploy.sh status    # 查看状态
```

**GitHub 仓库地址**：https://github.com/yinyayun/yuanmeng-odyssey.git

#### 方式二：手动部署

```bash
# 1. 克隆代码
git clone https://github.com/yinyayun/yuanmeng-odyssey.git /opt/yuanmeng-odyssey
cd /opt/yuanmeng-odyssey

# 2. 安装依赖
npm run install:all

# 3. 构建前端
cd frontend && npm run build

# 4. 配置 Nginx（参考 deploy.sh nginx 输出）

# 5. 启动后端
cd ../backend
pm2 start src/app.js --name yuanmeng-odyssey-backend
```

## 模块详解

### 时光银行

#### 赚取积分
1. 进入「积分存取」页面
2. 选择事项（如：写作业、阅读、运动）
3. 输入投入时间（分钟）
4. 系统自动计算可获得的积分
5. 点击「确认存入」

#### 消费积分
1. 在「积分存取」页面切换到提取
2. 选择想要的活动（如：看电视、玩游戏）
3. 输入想要的时间
4. 系统显示需要扣除的积分
5. 确认余额充足后点击「确认提取」

#### 兑换规则配置
- 在「兑换规则」页面可以添加/删除规则
- 赚取规则：X分钟 = Y积分
- 消费规则：X积分 = Y分钟

### 智慧树洞

1. 将成长计划 HTML 文件放入 `backend/uploads/growth_plans/` 目录
2. 按学期/日期组织文件夹结构
3. 在系统中按目录树浏览
4. 点击文件即可预览内容

示例结构：
```
growth_plans/
├── 4年级第二学期/
│   ├── 4年级第二学期成长计划总览.html
│   └── 20260405期内容.html
└── 5年级第一学期/
    └── ...
```

### 数能充电站

1. 将数学试卷 PDF/Word 放入 `backend/uploads/math_papers/` 目录
2. 在系统中浏览文件树
3. 支持在线预览 PDF
4. 点击下载按钮保存到本地

## 数据备份

### 自动备份
系统支持两种备份方式：

1. **GitHub 同步**：数据库自动同步到 GitHub
2. **本地备份**：保留最近 10 个备份文件

### 手动备份
```bash
# 本地备份
curl -X POST http://localhost:3001/api/backup/local

# GitHub 同步
curl -X POST http://localhost:3001/api/backup/git-sync
```

## 常见问题

### 1. 如何修改默认账户名称？
在数据库中修改 `accounts` 表的 `name` 字段，或重新创建账户。

### 2. 积分计算错误？
检查「兑换规则」中的时间和积分比例设置是否正确。

### 3. 文件上传后看不到？
确保文件放在正确的目录下，并刷新页面。

### 4. 如何重置数据？
删除 `backend/database/yuanmeng.db` 文件，重启服务会自动重建。

### 5. 如何更新到最新版本？
```bash
# 方式一：使用部署脚本（推荐）
sudo bash deploy.sh update

# 方式二：手动更新
cd /opt/yuanmeng-odyssey
git pull
npm run install:all
cd frontend && npm run build
pm2 restart yuanmeng-odyssey-backend
```

### 6. 如何修改默认密码？
1. 使用家长账号登录（dad 或 mom）
2. 进入管理后台 → 系统设置
3. 点击"修改管理员密码"
4. 输入旧密码和新密码

### 7. 登录问题排查
如果无法登录：
1. 检查后端服务是否运行：`sudo bash deploy.sh status`
2. 检查数据库是否初始化（查看 `/opt/yuanmeng-odyssey/backend/database/` 目录）
3. 查看后端日志：`sudo bash deploy.sh logs`
4. 确认使用正确的用户名和密码（注意大小写）

### 8. 部署失败怎么办？
1. 检查服务器是否能访问 GitHub
2. 检查是否有 root 权限
3. 查看日志：`sudo bash deploy.sh logs` 或 `pm2 logs yuanmeng-odyssey-backend`
4. 检查 Nginx 配置：`nginx -t`
5. 查看 Nginx 配置参考：`sudo bash deploy.sh nginx`
